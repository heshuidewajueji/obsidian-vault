# 深度研究命令

## 任务
对用户指定的主题进行系统性深度研究：
1. 先搜索 Vault 内已有的相关笔记
2. 结合网络搜索获取最新信息
3. 整合内外知识，形成全面分析

## 执行步骤

### Phase 1: Vault 内搜索
- 使用 grep 搜索 Vault 中与主题相关的笔记
- 识别相关的 MOC（内容地图）和索引笔记
- 汇总 Vault 内的现有观点

### Phase 2: 网络研究
- 搜索该主题的最新文章、论文、观点
- 识别不同的学派和立场
- 找出现有研究的空白

### Phase 3: 整合输出
- 将 Vault 知识与新研究结合
- 识别隐藏的联系和矛盾
- 生成可执行的洞察

## 输出格式

```markdown
# [主题] 深度研究报告

## Vault 内现有知识
[已掌握的知识点]

## 最新研究发现
[网络搜索获取的信息]

## 整合洞察
### 新发现
### 与 Vault 知识的连接
### 观点冲突点

## 行动建议
- 下一步研究建议
- 可发布的文章方向
- 需要验证的假设
```

## 触发方式
/research-deep-dive [主题]
