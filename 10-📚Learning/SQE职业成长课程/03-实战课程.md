## 课程定位

**目标人群**：完成基础课程（01-基础课程.md），具备一定实战经验，需要积累真实问题处理能力的SQE

**学习目标**：通过真实案例学习，快速建立问题处理能力，避免常见错误

**前置要求**：完成01-基础课程.md，理解8D、FMEA、VDA 6.3基础概念

## 案例学习方法论

### King的特征适配

- 性格：ISTP（务实动手型）
- 偏好：边做边学，从具体案例中提炼规律
- 策略：案例分析 → 提炼模板 → 迁移应用

### 费曼学习法应用

1. **读案例**：理解问题描述、根因分析、纠正措施
2. **讲案例**：用自己的话向非专业人士解释
3. **找差距**：对比自己的理解和案例处理方式的差异
4. **写复盘**：提炼可复用的检查清单和话术

## 第一部分：来料质量问题案例（5个）

### 案例1：方向盘传感器来料不良

**基本信息**：
- 供应商：XXX供应商
- 零件：方向盘角度传感器
- 问题：来料不良率连续2周超过1000ppm
- 影响：产线停线风险

**问题描述**：
```
问题：传感器来料不良率高达1500ppm，超出标准500ppm
何地：XXX供应商/XXX生产线
何时：2026-XX-XX首次发现，连续2周
何事：不良表现为焊接不良，导致功能失效
程度：当前不良率1500ppm，标准值500ppm，影响2条生产线
```

**根因分析（5Why）**：
```
Why1：为什么不良率高？因为焊接不良
Why2：为什么焊接不良？因为焊锡膏活性不足
Why3：为什么活性不足？因为库存超期（存放超6个月）
Why4：为什么库存超期？因为没有先进先出管理
Why5：为什么没有先进先出？因为仓库没有FIFO制度
根因：仓库缺少FIFO管理制度，焊锡膏超期使用
```

**纠正措施**：
- 立即：全检库存焊锡膏，隔离超期材料
- 短期：建立仓库FIFO制度，设置库存周期预警
- 长期：供应商引入物料管理系统

**经验教训**：
- 来料检验发现的往往是结果，真正问题在过程
- 库存管理是基础管理，基础不牢地动山摇

### 案例2：制动卡钳来料外观不良

**基本信息**：
- 供应商：XXX供应商
- 零件：制动卡钳总成
- 问题：来料外观划伤不良率8%
- 影响：装配线返工率上升

**问题描述**：
```
问题：卡钳来料外观划伤不良率8%，标准<1%
何地：XXX供应商
何时：2026-XX-XX
何事：表面划伤深度>0.2mm，影响涂装和装配
程度：不良率8%，返工率5%，影响装配效率
```

**根因分析**：
```
Why1：为什么外观不良率高？因为周转过程防护不足
Why2：为什么防护不足？因为没有专用防滑托盘
Why3：为什么没有防滑托盘？因为成本考虑，使用通用托盘
Why4：为什么用通用托盘？因为没有定义专用包装标准
Why5：为什么没有定义？因为APQP阶段包装验证不充分
根因：供应商APQP阶段包装设计验证不充分
```

**纠正措施**：
- 立即：对当前批次全检，隔离不良品
- 短期：设计专用防滑托盘，更新包装规范
- 长期：将包装设计纳入PPAP审核检查表

**经验教训**：
- 包装设计是APQP的重要输出，但常被忽视
- 制动安全件的外观要求更严格，需在审核中重点关注

### 案例3：底盘电控单元信号干扰

**基本信息**：
- 供应商：XXX供应商
- 零件：底盘电控单元（VCU）
- 问题：来料EMC测试信号干扰超标
- 影响：整车公告测试失败

**问题描述**：
```
问题：VCU来料EMC测试信号干扰超标3dB
何地：实验室测试
何时：2026-XX-XX
何事：辐射干扰和传导干扰均超标，影响整车公告
程度：公告测试失败，影响车辆上市时间节点
```

**根因分析**：
```
Why1：为什么EMC超标？因为PCB接地设计缺陷
Why2：为什么接地设计缺陷？因为设计评审未覆盖EMC风险
Why3：为什么未覆盖？因为供应商缺乏EMC专业能力
Why4：为什么缺乏EMC能力？因为从未做过系统性培训
Why5：为什么没培训？因为主机厂未提出明确要求
根因：供应商EMC能力不足，主机厂APQP阶段未做EMC专项审核
```

**纠正措施**：
- 立即：供应商紧急整改PCB设计，重新送样
- 短期：建立VCU专项EMC测试规范，纳入PPAP
- 长期：对电控供应商增加EMC能力审核

**经验教训**：
- 电控产品的EMC问题是系统性设计问题，8D只能救火
- APQP阶段必须加入电控专项审核，否则后患无穷

### 案例4：转向系统螺栓松动

**基本信息**：
- 供应商：XXX供应商
- 零件：转向横拉杆总成
- 问题：螺栓扭矩衰减导致松脱
- 影响：转向系统安全风险

**问题描述**：
```
问题：横拉杆螺栓扭矩衰减超过30%，存在松脱风险
何地：整车装配后路试
何时：2026-XX-XX路试发现
何事：扭矩从设计值10N·m衰减到7N·m以下
程度：安全相关问题，立即启动召回风险评估
```

**根因分析**：
```
Why1：为什么扭矩衰减？因为螺栓未使用防松垫圈
Why2：为什么未使用防松垫圈？因为设计图纸未明确
Why3：为什么图纸未明确？因为设计工程师遗漏
Why4：为什么遗漏？因为没有FMEA交叉审核
Why5：为什么没有交叉审核？因为供应商流程缺失设计验证环节
根因：供应商设计验证流程不完整，FMEA未覆盖螺栓防松风险
```

**纠正措施**：
- 立即：排查已装车车辆，重新拧紧并加防松垫圈
- 短期：更新设计图纸，明确防松要求，提交PPAP变更
- 长期：建立设计FMEA交叉审核机制

**经验教训**：
- 安全相关件的设计必须做FMEA交叉审核
- 扭矩衰减是渐进过程，需建立过程扭矩监控（SPC）

### 案例5：线束来料PIN脚氧化

**基本信息**：
- 供应商：XXX供应商
- 零件：仪表线束总成
- 问题：来料PIN脚氧化，不良率3%
- 影响：整车电气功能失效风险

**问题描述**：
```
问题：线束连接器PIN脚氧化，不良率3%
何地：XXX供应商
何时：2026-XX-XX
何事：PIN脚镀层氧化，导致接触不良
程度：电气功能失效，影响售后质量投诉
```

**根因分析**：
```
Why1：为什么PIN脚氧化？因为镀层厚度不足
Why2：为什么镀层厚度不足？因为电镀工艺参数偏移
Why3：为什么工艺参数偏移？因为电镀设备老化未维护
Why4：为什么设备未维护？因为没有预防性维护计划
Why5：为什么没有维护计划？因为供应商质量管理意识薄弱
根因：供应商设备预防性维护缺失，导致电镀质量不稳定
```

**纠正措施**：
- 立即：全检库存线束，增加镀层厚度抽检
- 短期：要求供应商建立设备预防性维护计划
- 长期：将设备维护记录纳入审核必查项

**经验教训**：
- 电镀质量是基础工艺质量，影响电气连接可靠性
- 设备维护是过程能力的保障，必须纳入供应商管理

**审核发现记录**：

| 序号 | 审核提问 | 证据来源 | 判定 | 备注 |
|------|----------|----------|------|------|

## 第三部分：跨部门协调案例（2个）

### 案例1：供应商质量升级推动

**背景**：
- 问题：某关键零件供应商不良率突然上升至2000ppm
- 影响：影响2个车型生产
- 僵局：供应商内部意见不统一，改进措施推进缓慢

**协调过程**：

| 步骤 | 行动 | 结果 |
|------|------|------|
| 1 | 组织三方会议（主机厂SQE+供应商质量+供应商技术） | 统一认识，明确各方责任 |
| 2 | 制定联合改进计划 | 确定3个关键改进点，3周时间节点 |
| 3 | 建立日报机制 | 每日同步进展，及时识别障碍 |
| 4 | 推动供应商高层介入 | 供应商总经理主持周例会 |
| 5 | 引入外部资源 | 联系设备供应商技术支持 |

**关键成功因素**：
- 用数据说话（不良率趋势图、影响分析）
- 明确时间节点和责任人
- 让供应商高层直接参与

## 第四部分：控制论诊断案例

### 综合诊断：供应商质量危机控制论分析

**问题**：某转向供应商连续3个月质量下滑，不良率从300ppm升至1500ppm

**控制论五层诊断**：

| 层级 | 诊断 | 发现 |
|------|------|------|
| L1执行层 | 日常检验执行正常，但深层问题隐蔽 | 发现抽检比例偏低，关键特性未全覆盖 |
| L2结构层 | 组织架构稳定，但职能分工有缝隙 | 质量部与技术部沟通不畅，信息传递断层 |
| L3控制层 | 控制机制失效，无效纠正措施反复执行 | 8D写了8份，根因分析停留在表面层 |
| L4信息层 | 信息采集不完整，KPI体系缺失 | 只有不良率数据，缺少过程能力数据 |
| L5目标层 | 目标设定模糊，与主机厂期望有偏差 | 供应商认为"还行"，主机厂认为"不可接受" |

**系统干预方案**：

```
L5层干预：重新对齐质量目标（从"合格"到"零缺陷"）
    ↓
L4层干预：建立完整KPI体系（不良率+Cpk+响应速度）
    ↓
L3层干预：重新审核8D有效性，实施根本纠正措施
    ↓
L2层干预：优化跨部门沟通机制，建立联合质量会议
    ↓
L1层干预：提高抽检比例，增加关键特性检测
```
