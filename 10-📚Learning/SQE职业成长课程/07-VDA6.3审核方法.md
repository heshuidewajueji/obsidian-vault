## 一、VDA 6.3概述

### 1.1 标准背景

**全称**：VDA 6.3 - 过程审核（Process Audit）
**发布机构**：德国汽车工业协会（VDA）
**适用范围**：汽车行业供应商过程审核
**版本**：VDA 6.3:2023（第3版）

### 1.2 VDA 6.3:2023关键变化

| 变化项 | 说明 |
|--------|------|
| 软件方面纳入审核 | 新增软件过程相关提问 |
| A-SPICE整合 | 与VDA MLA协调，参考A-SPICE 3.1 |
| 提问精简 | 由原来59个提问精简为35个 |
| P3/P4增加采购要求 | 产品和过程开发中强化供应商采购管理 |
| 审核员资质规范化 | 明确审核员培训和知识要求 |
| 远程审核说明 | 增加远程审核实施指南 |

## 二、七大过程要素（P1-P7）

### 2.1 过程要素总览

```
P1: 潜在供应商分析（Potential Supplier Analysis）
    ↓ 覆盖P2-P7所有过程要素（潜在供应商准入审核）
P2: 项目管理（Project Management）
P3: 产品和过程开发的策划（Product and Process Development Planning）
P4: 产品和过程开发的实现（Product and Process Development Realization）
P5: 供应商管理（Supplier Management）
P6: 过程分析/生产（Process Analysis / Production）
P7: 顾客关怀、顾客满意、服务（Customer Care, Customer Satisfaction, Service）
```

### 2.2 P2 项目管理

**核心提问领域**（P2.1子要素）：
- 是否建立了项目管理战略（含项目组织机构）
- 是否存在项目管理过程定义
- 是否确定了跨学科的项目组织机构（CFT）
- 是否规定了项目团队成员的职责、能力和权限

**CFT核心成员**：
- PM（项目经理）
- 产品质量技术人员/开发人员
- 质量人员/SQE
- 采购人员
- 工艺、生产、设备开发人员
- 物流人员
- 销售支持人员

**审核要点**：
- 项目组织机构图是否已建立
- 职责矩阵（RACI）是否清晰
- 项目进度是否受控（APQP执行情况）
- 跨职能小组是否有效运作

### 2.3 P3 产品和过程开发的策划

**D-FMEA（设计FMEA）审核**：
- 是否进行了设计FMEA
- 是否确定了改进措施
- FMEA是否在项目过程中更新
- 已确定措施是否落实

**P-FMEA（过程FMEA）审核**：
- 是否进行了过程FMEA
- 关键特殊特性是否识别
- 高RPN项是否有改进计划

**质量策划**：
- 控制计划是否已制定
- 是否覆盖从进料到出货全流程
- 检验计划是否与FMEA对应

### 2.4 P4 产品和过程开发的实现

**过程开发审核**：
- 过程FMEA是否动态更新
- 质量计划是否落实
- 试生产是否按计划执行
- 生产文件/检验文件是否齐全有效

**批量生产认可**：
- 是否在批量生产条件下进行了试生产
- 初始过程能力是否达标（Cpk≥1.33）
- MSA是否完成且GRR<10%
- PPAP是否已提交并获批准

### 2.5 P5 供应商管理

**核心审核内容**：
- 供应商选择过程是否有记录
- 供应商FMEA/控制计划是否审查
- 供应商审核计划是否执行
- 来料检验规程是否有效
- 不合格品处理流程是否规范

**SQE日常关注**：
- 供应商是否对分供方进行了有效管理
- 供应商变更是否及时通知并重新批准
- 来料不良数据是否统计并反馈

### 2.6 P6 过程分析/生产（生产过程审核）

**P6子要素分解**：

| 子要素 | 审核内容 |
|--------|----------|
| P6.1 过程输入 | 原材料/来料检验、工装设备状态 |
| P6.2 过程流程 | 作业指导书执行、5S状态 |
| P6.3 人力资源 | 人员资质、培训记录、技能矩阵 |
| P6.4 物质资源 | 设备点检、工装维护、测量系统 |
| P6.5 效果、效率、避免浪费 | 过程能力、不合格率、持续改进 |
| P6.6 过程输出 | 出货检验、客户投诉、文件记录 |

**5M1E审核法**：
- Man（人）：操作员是否培训合格、是否持证
- Machine（机）：设备点检、维护记录
- Material（料）：来料检验、物料状态
- Method（法）：作业指导书版本、执行情况
- Measurement（测）：量具校准、MSA有效性
- Environment（环）：温湿度、静电防护、清洁度

### 2.7 P7 顾客关怀、顾客满意、服务

**核心内容**：
- 顾客反馈处理流程是否有效
- 顾客投诉是否有记录和闭环
- 8D报告是否按时提交
- 持续改进措施是否落实
- 顾客满意度是否定期评估

## 三、VDA 6.3审核实施流程

### 3.1 审核准备阶段

**审核计划内容**：
- 审核目的和范围
- 审核依据（标准条款）
- 审核组人员（至少2人）
- 审核时间表
- 受审核方联系人

**收集资料**：
- 供应商基本信息（组织架构、产品类别）
- 最近一次审核报告
- 质量数据（来料不良率、过程不良率）
- 顾客投诉记录
- FMEA和控制计划

**审核检查表设计**：
- 按P1-P7要素编制提问清单
- 每条提问明确：证据来源、判定标准
- 标注适用性（适用/不适用）

### 3.2 现场审核阶段

**过程研究方法**：
- 沿增值链（从进料到出货）开展审核
- 假定各过程输入已确定
- 分析输入对后续过程的影响

**现场观察要点**：

| 观察点 | 关注内容 | 异常信号 |
|--------|----------|----------|
| 5S | 现场清洁、物料定位 | 杂乱、油污、地面不洁 |
| 工装设备 | 点检记录、设备状态 | 点检缺失、设备故障 |
| 在制品 | 流转状态、半成品标识 | 无标识、混放 |
| 返工品 | 处理流程、区域隔离 | 随意放置、无记录 |
| 作业文件 | 版本有效性、记录完整性 | 版本过期、空白记录 |
| 人员 | 操作是否符合作业指导书 | 与SOP不符 |
| 测量系统 | 量具校准状态、MSA报告 | 超期校准、无MSA |

### 3.3 审核评价

**评价标准（符合率计算）**：
```
符合率 = (符合项数 / 应审核项数) × 100%
```

**定级标准**：

| 等级 | 描述 | 符合率要求 |
|------|------|-----------|
| A级 | 优秀 | ≥90% |
| B级 | 良好 | 75%-89% |
| C级 | 合格 | 60%-74% |
| D级 | 不合格 | <60% |

**重要规则**：
- 当某一要素的符合率低于75%时，必须降级
- 软件相关提问不适用时，标注n.a/n.b
- 每一要素或子要素审核提问至少达到2/3以上，方可确认审核有效

## 四、VDA 6.3提问表（P6生产过程详细提问）

### 4.1 P6.1 过程输入

| 提问 | 证据来源 | 判定 |
|------|----------|------|
| 进料检验是否按规定执行 | 进料检验记录、检验标准 | OK/NOK |
| 来料不良是否隔离并标识 | 不良品区、现场状态 | OK/NOK |
| 原材料是否在有效期内 | 物料标签、批次管理 | OK/NOK |
| 工装设备是否有点检记录 | 点检表、最近3个月 | OK/NOK |

### 4.2 P6.2 过程流程

| 提问 | 证据来源 | 判定 |
|------|----------|------|
| 是否有作业指导书（SOP） | 现场文件、版本有效性 | OK/NOK |
| 操作是否按SOP执行 | 现场观察、录像抽查 | OK/NOK |
| 关键特性是否有控制图 | SPC控制图、异常记录 | OK/NOK |
| 不合格品处理是否按流程 | 不良品处理单、隔离区 | OK/NOK |

### 4.3 P6.3 人力资源

| 提问 | 证据来源 | 判定 |
|------|----------|------|
| 操作员是否经过培训 | 培训记录、上岗证 | OK/NOK |
| 特殊工序是否持证上岗 | 证书、有效期 | OK/NOK |
| 培训记录是否存档 | 人事档案、培训签到表 | OK/NOK |

### 4.4 P6.4 物质资源

| 提问 | 证据来源 | 判定 |
|------|----------|------|
| 测量系统是否经过MSA | MSA报告、GRR<10% | OK/NOK |
| 量具是否在校准期内 | 校准证书、校准标签 | OK/NOK |
| 设备维护保养是否按计划 | 维护记录、点检表 | OK/NOK |

### 4.5 P6.5 效果、效率、避免浪费

| 提问 | 证据来源 | 判定 |
|------|----------|------|
| 关键特性Cpk是否达标 | SPC报告、Cpk≥1.33 | OK/NOK |
| 不良品率是否有改善趋势 | 质量数据趋势图 | OK/NOK |
| 是否有持续改进项目 | 改善提案、QC小组 | OK/NOK |

### 4.6 P6.6 过程输出

| 提问 | 证据来源 | 判定 |
|------|----------|------|
| 出货检验是否按规定执行 | 出货检验记录、AQL | OK/NOK |
| 顾客投诉是否及时处理 | 8D报告、响应时间 | OK/NOK |
| 文件记录是否完整可追溯 | 批次记录、追溯系统 | OK/NOK |

## 五、审核发现分级

### 5.1 发现分类

| 分类 | 说明 | 示例 |
|------|------|------|
| MAJOR | 重大不符合 | 缺失FMEA、关键特性无控制、重复性问题未解决 |
| MINOR | 轻微不符合 | 点检记录不及时、文件版本未更新 |
| OBS | 改进建议 | 5S可提升、建议增加防错措施 |
| OK | 符合 | 无异常发现 |

### 5.2 升级条件

**满足以下任一条件必须升级处理**：
1. 发现项数量超过检查表总数20%
2. 同一问题重复出现（整改后又发生）
3. 涉及安全/法规要求的关键项不符
4. 供应商质量数据持续恶化
5. 顾客投诉未在规定时间内关闭

## 六、VDA 6.3与IATF 16949关系

### 6.1 审核类型对比

| 审核类型 | 关注点 | 覆盖范围 |
|----------|--------|----------|
| VDA 6.3过程审核 | 过程能力和有效性 | P1-P7全流程 |
| IATF 16949体系审核 | 质量管理体系完整性 | 全部过程/部门 |
| 制造过程审核 | 单个工序有效性 | 5M1E |

### 6.2 VDA 6.3在IATF 16949中的应用

| IATF 16949条款 | 对应VDA 6.3要素 |
|----------------|-----------------|
| 第4章 质量管理体系 | P2项目管理 |
| 第8.1 运行策划和控制 | P3/P4产品和过程开发 |
| 第8.2.1 顾客沟通 | P7顾客关怀 |
| 第8.4 外部提供过程、产品和服务控制 | P5供应商管理 |
| 第8.5.1 生产和服务提供 | P6过程分析/生产 |
| 第10.2 不合格和纠正措施 | P7顾客投诉处理 |

## 七、SQE审核实战清单

### 审核前准备
- [ ] 确认审核目的和范围
- [ ] 收集供应商基本信息（组织架构、产品、工艺）
- [ ] 调阅上次审核报告和质量数据
- [ ] 准备检查表（含提问清单和判定标准）
- [ ] 确认审核组成员和分工

### 审核实施
- [ ] 首次会议（审核组与供应商管理层沟通）
- [ ] P2项目管理审核（项目组织、FMEA状态）
- [ ] P3/P4开发审核（APQP执行、PPAP状态）
- [ ] P5供应商管理审核（来料检验、分供方管理）
- [ ] P6生产过程审核（5M1E全覆盖）
- [ ] P7顾客反馈审核（投诉处理、8D闭环）
- [ ] 现场观察记录（关键工序、特殊特性）

### 审核后
- [ ] 末次会议（与供应商管理层交换意见）
- [ ] 编写审核报告（24小时内）
- [ ] 发出不符合项（MINOR/MAJOR）
- [ ] 跟踪整改（规定时限内关闭）
- [ ] 审核资料存档

## 八、常见误区

| 误区 | 正确理解 |
|------|----------|
| VDA 6.3只适合德系OEM | VDA 6.3被全球汽车行业广泛认可，适用于所有供应商 |
| 审核就是找茬 | 审核是帮供应商发现改进机会，促进持续改进 |
| 检查表照搬不用改 | 检查表需根据供应商产品和工艺定制 |
| 只看文件不看现场 | 文件是必要条件，现场是充分条件，两者必须结合 |
| 审核完就结束 | 审核后必须跟踪整改，验证有效性 |

## 九、主动输出任务（L1）

### 任务一：VDA 6.3审核检查表编制

**任务描述**：为一个实际供应商编制完整的VDA 6.3审核检查表

**任务要求**：
1. 选择一个熟悉的供应商或假设的冲压件供应商
2. 按P1-P7要素编制提问清单，不少于30条提问
3. 每条提问需标注：证据来源、判定标准（OK/NOK）
4. 形成Word或Excel格式检查表

**提交物**：
- 完整的审核检查表（P1-P7）
- 每条提问的判定标准说明

**验收标准**：
- P6生产过程审核提问不少于15条
- 覆盖5M1E各要素
- 提问具有可操作性（能通过现场观察或文件审查验证）

---

### 任务二：模拟审核报告撰写

**任务描述**：基于以下模拟发现项，撰写一份完整的VDA 6.3审核报告

**模拟场景**：
某零部件供应商，审核发现以下问题：
- P3: FMEA未按项目节点更新（更新滞后3个月）
- P5: 分供方未进行现场审核，仅文件审查
- P6.2: 关键工序作业指导书版本过期（现行版本V2.0，文件记录为V1.0）
- P6.4: 量具校准证已过期1个月
- P6.5: 某关键特性Cpk=1.15（要求≥1.33）

**提交物**：
- 审核报告（含：审核概况、审核范围、审核发现、整体评价）
- 不符合项报告（至少2份MAJOR）

**验收标准**：
- 符合率计算正确
- 定级准确（B级或C级）
- 改进建议具体可操作

---

### 任务三：不符合项整改跟踪

**任务描述**：对一份供应商不符合项进行整改跟踪和验证

**练习内容**：
- 模拟供应商整改回复（包含：原因分析、纠正措施、预防措施）
- 评估整改回复的有效性
- 验证整改措施的实施证据

**验收标准**：
- 能识别整改回复中的不足（如：原因分析不深入、措施无法验证）
- 能提出有效的跟踪验证计划

## 十、Anki记忆卡（L2）

### 卡片组一：VDA 6.3基础概念

**Q1**: VDA 6.3的全称是什么？适用于什么范围？
**A1**: VDA 6.3全称"过程审核（Process Audit）"，适用于汽车行业供应商过程审核，由德国汽车工业协会（VDA）发布。

---

**Q2**: VDA 6.3:2023版本的主要变化有哪些？
**A2**:
- 软件方面纳入审核
- 与A-SPICE 3.1整合
- 提问由59个精简为35个
- P3/P4增加采购要求
- 审核员资质规范化
- 增加远程审核指南

---

**Q3**: VDA 6.3七大过程要素是什么？
**A3**: P1潜在供应商分析 → P2项目管理 → P3产品和过程开发策划 → P4产品和过程开发实现 → P5供应商管理 → P6过程分析/生产 → P7顾客关怀

---

### 卡片组二：P6生产过程审核

**Q4**: P6的六个子要素是什么？
**A4**: P6.1过程输入 → P6.2过程流程 → P6.3人力资源 → P6.4物质资源 → P6.5效果效率避免浪费 → P6.6过程输出

---

**Q5**: 5M1E审核法各指什么？
**A5**: Man（人-操作员资质）→ Machine（机-设备点检）→ Material（料-来料检验）→ Method（法-作业指导书）→ Measurement（测-测量系统）→ Environment（环-环境条件）

---

**Q6**: 初始过程能力Cpk要求是多少？MSA GRR要求是多少？
**A6**: Cpk≥1.33，MSA GRR<10%

---

### 卡片组三：审核评价

**Q7**: 符合率如何计算？A/B/C/D级对应的符合率是多少？
**A7**: 符合率=(符合项数/应审核项数)×100%
- A级≥90%（优秀）
- B级75%-89%（良好）
- C级60%-74%（合格）
- D级<60%（不合格）

---

**Q8**: 什么情况下必须降级？
**A8**: 当某一要素的符合率低于75%时，必须降级

---

**Q9**: MAJOR和MINOR不符合的区别是什么？各举2例
**A9**: MAJOR（重大不符合）：缺失FMEA、关键特性无控制、重复性问题未解决
