## 八、Skills 同步与管理

### 8.1 同步方式

```json
// settings.json
{
  "skillSyncMethod": "symlink",
  "skillStorageLocation": "cc_switch"
}
```

### 8.2 同步到 Hermes

```bash
# Skills 自动同步到
~/.king-ai/skills/

# Hermes 自动加载
```

### 8.3 备份与恢复

```bash
# 备份 Skills
cp -r ~/.cc-switch/skills/ ~/.cc-switch/skill-backups/skills_$(date +%Y%m%d)/

# 恢复
cp -r ~/.cc-switch/skill-backups/skills_YYYYMMDD/* ~/.cc-switch/skills/
```
