## 一、对话交互

### 1.1 基本对话

```bash
hermes
```

```
你：你好
Hermes：你好！有什么我可以帮你的？

你：帮我总结下今天的会议
Hermes：[根据上下文提供总结]

你：把这个发给小明
Hermes：[执行发送操作]
```

### 1.2 多轮对话

```
你：明天的日程是什么？
Hermes：明天 2026-05-23：
- 10:00 产品评审
- 14:00 团队周会

你：帮我设置 10 点前 15 分钟的提醒
Hermes：好的，已为你设置提醒。

你：把这两个都记到我的日程表里
Hermes：[更新日程]
```

## 二、持久记忆系统

### 2.1 自动记忆

Hermes 会自动记住：
- 用户偏好
- 对话历史
- 重要信息
- 任务状态

```
你：记住我的咖啡偏好是拿铁加一份浓缩
Hermes：已记住，你喜欢拿铁加一份浓缩。

你：以后提到咖啡就用这个偏好
Hermes：好的，我会记住。
```

### 2.2 记忆查询

```
你：我之前告诉你我喜欢什么咖啡？
Hermes：根据我的记忆，你喜欢拿铁加一份浓缩。
```

### 2.3 记忆管理

```bash
# 查看记忆存储位置
ls ~/.hermes/memory/

# 手动编辑记忆（高级）
hermes memory edit
```

## 三、技能系统

### 3.1 内置技能

| 技能 | 说明 |
|------|------|
| **web_search** | 网络搜索 |
| **calculator** | 数学计算 |
| **reminder** | 设置提醒 |
| **schedule** | 日程管理 |
| **file_read** | 读取文件 |
| **file_write** | 写入文件 |

### 3.2 技能调用

```
你：帮我搜索一下今天的天气
Hermes：[调用 web_search 技能]
结果是：北京今天晴，25-30°C

你：10 分钟后提醒我开会
Hermes：[调用 reminder 技能]
好的，已设置提醒。
```

### 3.3 自定义技能

```bash
# 创建技能
hermes skill create my-skill

# 查看可用技能
hermes skill list

# 删除技能
hermes skill delete my-skill
```

### 3.4 技能目录

```
~/.hermes/skills/
├── web_search/
│   └── skill.yaml
├── calculator/
│   └── skill.yaml
└── my-custom-skill/
    └── skill.yaml
```

## 五、文件操作

### 5.1 读取文件

```
你：读取桌面的 notes.txt 文件
Hermes：文件内容：
[读取并显示文件内容]
```

### 5.2 写入文件

```
你：帮我创建一个 todo.md 文件，包含今天的任务
Hermes：[创建文件]
已创建 ~/todo.md
```

### 5.3 文件搜索

```
你：在我的文档里搜索包含"会议"的文件
Hermes：找到以下文件：
- ~/Documents/会议纪要/2026-05-20.md
- ~/Documents/会议记录.docx
```
