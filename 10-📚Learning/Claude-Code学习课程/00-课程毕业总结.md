## 🏆 核心技能掌握

### 1. 基础操作
- ✅ 安装 Claude Code (`npm install -g @anthropic-ai/claude-code`)
- ✅ 配置 API (Volcengine)
- ✅ 启动和使用 (`claude`)

### 2. 指令系统
- ✅ `/` 控制指令 (help, plan, model, approve)
- ✅ `@` 引用文件/目录
- ✅ `#` 注释标记
- ✅ `?` 搜索历史

### 3. 工具使用
- ✅ Read/Write/Edit 文件操作
- ✅ Glob/Grep 搜索
- ✅ Bash 命令执行
- ✅ WebSearch/WebFetch 网络工具

- ✅ CLAUDE.md 项目规范
- ✅ memory.md 跨会话记忆

- ✅ skills/ 可复用技能

### 5. 进阶技能
- ✅ 调试排错：日志分析、权限配置
