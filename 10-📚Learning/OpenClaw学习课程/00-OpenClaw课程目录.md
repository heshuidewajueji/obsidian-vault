## 课程概述

OpenClaw 是一个开源的 **AI Agent 编排框架**，支持多 Agent 管理、跨平台消息通道、统一记忆存储。

**核心定位**：统一的 Agent 管理与协作平台

## OpenClaw vs 其他 Agent

| 维度 | OpenClaw | Hermes | Claude Code |
|------|----------|--------|-------------|
| **定位** | Agent 编排/协作 | 个人助手/自进化 | 编程助手 |
| **多 Agent** | ✅ 原生支持 | ❌ | ❌ |
| **记忆共享** | ✅ Vestige 向量存储 | ❌ | ❌ |
| **通道集成** | ✅ 多平台 | ✅ 多平台 | ❌ |
| **编程能力** | 一般 | 一般 | ✅ 强大 |
| **自进化** | ❌ | ✅ | ❌ |
