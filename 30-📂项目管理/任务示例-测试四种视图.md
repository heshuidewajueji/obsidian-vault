# 任务示例：测试四种视图

## 📌 任务描述

在 Projects 插件中测试 Table / Board / Calendar / Gallery 四种视图。

## ✅ 完成标准

- [x] Table 视图：添加新列、排序、筛选
- [x] Board 视图：拖拽卡片更新状态
- [x] Calendar 视图：查看任务时间线
- [x] Gallery 视图：添加封面图片

## 📋 测试步骤

### 1. Table 视图测试

1. 打开 Projects 插件
2. 选择 Table 视图
3. 添加"负责人"列
4. 按优先级排序

### 2. Board 视图测试

1. 切换到 Board 视图
2. 将任务从 "待处理" 拖到 "进行中"
3. 观察 frontmatter 中状态的变化

### 3. Calendar 视图测试

1. 切换到 Calendar 视图
2. 查看本月到期的任务
3. 测试点击日期创建新任务

### 4. Gallery 视图测试

1. 切换到 Gallery 视图
2. 为任务笔记添加封面图片
3. 查看卡片封面效果
